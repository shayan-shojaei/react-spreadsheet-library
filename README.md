# React Spreadsheet

A Spreadsheet component library for React!

## Scripts

### Build

```bash
yarn build
```

### Start Storybook server

```bash
yarn dev
```

### Run tests

```bash
yarn test
```

### Run and watch tests

```bash
yarn test-watch
```

### Check for linting issues

```bash
yarn lint
```

### Check for formatting issues

```bash
yarn format-check
```

### Fix formatting issues

```bash
yarn format
```
